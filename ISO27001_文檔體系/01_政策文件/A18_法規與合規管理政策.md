## 1. 目的
確保公司遵守所有適用的法律、法規與合約要求，建立合規管理機制，降低法律與運營風險。

## 2. 適用法律與法規

### 2.1 台灣相關法規
| 法規 | 適用項目 | 責任單位 |
|------|---------|---------|
| **個人資料保護法** | 客戶、員工個資保護 | 人資、資安 |
| **電子簽章法** | 電子合約、簽署 | 法務、IT |
| **資訊安全管理法** | 關鍵基礎設施保護 | 資安 |
| **勞動基準法** | 員工工時、福利 | 人資 |
| **所得稅法** | 薪資、稅務申報 | 財務 |
| **營業稅法** | 進銷項報稅 | 財務 |
| **公司法** | 股東會、董事會紀錄 | 法務 |
| **商業會計法** | 帳簿記錄保存 | 財務 |
| **運輸安全相關** | 危險物品運輸、駕駛時數 | 營運 |

### 2.2 國際標準與框架
- **ISO/IEC 27001:2022**：資訊安全管理系統（本公司目標）
- **GDPR**：若處理歐盟個人資料
- **SOC 2**：若與國際企業合作
- **ISO 22301**：業務連續性管理

### 2.3 行業標準
- **物流業安全運輸標準**
- **供應鏈安全標準**（客戶要求）
- **環保與勞工標準**

## 3. 合規識別與評估

### 3.1 法規責任人
- **法務部門**：追蹤法律變動，評估影響
- **資安部門**：評估資訊安全法規需求
- **人資部門**：勞工與隱私法規合規
- **財務部門**：稅務與會計法規合規
- **營運部門**：運輸與安全法規合規

### 3.2 年度合規評估
- **1月**：更新法規清單（含新增與修訂）
- **3月**：各部門自評合規狀況
- **6月**：資安進行獨立評估
- **9月**：提出改進計畫與期限
- **12月**：驗證改進落實情況

### 3.3 合規評估內容
- 政策與程序是否符合法規
- 控制措施是否有效實施
- 記錄與證據是否完整保存
- 員工培訓與意識是否足夠
- 發現問題與改進進度

## 4. 資訊安全法規重點

### 4.1 個人資料保護
- **定義**：可識別自然人身份的資訊（姓名、身分證號、電話、住址等）
- **合法基礎**：蒐集個資需明確告知目的，獲得同意
- **安全保護**：採取技術與組織措施保護個資（加密、存取控制等）
- **權利保障**：提供存取、更正、刪除權利
- **洩露通報**：72小時內通報主管機關與個人
- **留存期限**：依法規或業務需求，不超過必要期限
- **責任**：資訊管理者與處理者均應簽訂書面契約

### 4.2 資訊安全責任
- 採取適當安全措施（技術與管理）
- 定期進行風險評估與稽核
- 建立應急事件管理流程
- 員工安全教育與培訓
- 維護完整的審計日誌

### 4.3 合作廠商合規
- 簽訂個資處理契約
- 確保供應商採取同等安全措施
- 定期稽核供應商合規狀況
- 違規時可要求改善或終止合約

## 5. 稽核與檢查

### 5.1 內部稽核
- **對象**：所有部門與程序
- **頻率**：重點流程每年不少於2次
- **執行**：資安部門獨立執行，財務部門監督
- **報告**：稽核報告提交管理層與董事會
- **追蹤**：發現事項須有改進計畫與進度追蹤

### 5.2 外部稽核
- **監管檢查**：定期接受監管單位檢查
- **認証稽核**：ISO 27001認証每3年進行1次，年度監查
- **客戶稽核**：大客戶可要求進行合規稽核
- **第三方審計**：法律與稅務顧問定期審計

### 5.3 稽核發現與改善
- 稽核發現分為Critical/Major/Minor等級
- Critical與Major事項需於30天內改善
- Minor事項需於60天內改善
- 改善後需進行複查驗證
- 所有記錄保存≥3年

## 6. 違規與制裁

### 6.1 行政罰款
| 違規項目 | 主管機關 | 罰款範圍 |
|---------|---------|---------|
| 個資未妥善保護 | 個資委員會 | 5K-NT$50M |
| 未通報個資洩露 | 個資委員會 | 10K-NT$50M |
| 員工違反政策 | 公司內部 | 警告-解僱 |
| 供應商違反條款 | 合約追訴 | 違約金-終止 |

### 6.2 法律後果
- 民事賠償：被害人可提告索賠
- 刑事責任：嚴重違規可追究刑事
- 聲譽損害：媒體報導造成的品牌傷害
- 客戶影響：失去客戶信任，合約終止

### 6.3 內部處置
- 違規員工需進行紀律處分
- 管理層失職需追究責任
- 建立改正與預防措施
- 保存事件紀錄≥3年

## 7. 例外與豁免

### 7.1 申請程序
- 無法立即遵守時提出豁免申請
- 說明原因、影響、補強措施、改善期限
- 由相應管理層與資安核准
- 設定複查時間點

### 7.2 風險評估
- 違規帶來的風險評估
- 補強措施的有效性評估
- 決定是否批准豁免
- 豁免期限不超過6個月

## 8. 法令遵循合約

### 8.1 員工簽署
- 新進員工入職時簽署政策確認書
- 確認已閱讀相關政策與規定
- 保存簽署紀錄≥員工在職期間+2年

### 8.2 供應商合約
- 供應商簽署條款同意書
- 包含機密、安全、隱私、法規條款
- 違反時可終止合約

## 9. 文件與記錄保留

### 9.1 保留期限

| 文件類型 | 保留期限 | 保存地點 |
|---------|---------|---------|
| 法律合約 | 合約終止後3年 | 安全櫃 |
| 稽核報告 | 永久 | 文檔系統 |
| 個資同意書 | 取得後3年 | 文檔系統 |
| 帳簿與發票 | 5年 | 財務部門 |
| 稅務文件 | 7年 | 財務部門 |
| 員工紀錄 | 離職後3年 | 人資部門 |
| 安全事件 | 3年 | 資安部門 |

### 9.2 保存方式
- 原件保存於防火櫃或金庫
- 副本存放於文檔系統（加密）
- 過期文件按銷毀程序銷毀
- 銷毀應有見證與證明

## 10. 持續改進

### 10.1 法令動態掌握
- 訂閱法律變動通知
- 定期與法律顧問溝通
- 參與業界論壇與研討會
- 與同業交流最佳實踐

### 10.2 政策更新
- 法規變動時評估影響
- 更新相關政策與程序（如本政策）
- 員工通知與培訓
- 變更記錄與版本控制
