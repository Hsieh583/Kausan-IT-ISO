# IT 資訊安全與營運整合管理專案

> 本專案為 **ISO/IEC 27001:2022 資訊安全管理系統（ISMS）** 與 **IT 服務營運管理** 之整合專案範本，涵蓋合規治理、營運框架、自動化工具、人員培訓等面向。

> ⚠️ **重要安全提醒**: 此為公開專案，所有敏感資訊已被移除或替換為通用佔位符。請參閱 [SECURITY.md](SECURITY.md) 了解安全使用指南。

---

## 一、專案目的與範圍

本專案為四大核心領域的統一管理平台：

### 📋 1. ISO 27001 合規治理
* 建立與維護 ISO 27001 所需之政策（Policy）、程序（Procedure）與治理文件
* 以「**最小可行文件集合（Minimum Viable ISMS）**」為原則，避免文件爆量
* 提供清楚、可追溯、可稽核的版本控制機制

### 🎯 2. IT 真實營運管理（L1-L5）
* 將安全控制項轉化為 IT 實際營運活動
* 建立 **5 層成熟度管理框架**（L1-L5），管理 7 項核心 IT 服務
* 定義人力、工具、預算與績效指標

### ⚙️ 3. IT 自動化與工具整合
* 開發 API Hook 工具，連接 Wazuh、AD、Veeam 等系統自動產生稽核證據
* 減少人工記錄負擔，提高數據完整性與實時性
* 實現「系統自動記錄，手動僅填補空白」之運營模式

### 📚 4. IT 人才培育與教育訓練
* 建立與 L1-L5 營運層級對應的技術與知識體系
* 提供技術詞彙、實操指南、案例研究
* 支持組織內部知識轉移與能力建設

**核心價值主張**：從「合規文件」→「營運框架」→「自動化工具」→「人才培育」之閉環管理

## 二、治理原則（必讀）

### 1. 文件類型定義

本專案僅允許以下四種類型文件：

| 類型                | 說明                            |
| ----------------- | ----------------------------- |
| **Policy（政策）**    | 管理承諾與原則，不含操作細節                |
| **Procedure（程序）** | 跨角色流程與責任定義                    |
| **SOP（作業指引）**     | 實際操作方式，**僅能以 Procedure 附錄存在** |
| **Record（記錄）**    | 已發生事實之證據（僅定義，不保存實體）           |

❗ **禁止新增其他文件類型**（如 Guideline、Manual、Note 等）。

---

### 2. SOP 附錄化（強制規則）

* SOP **不得獨立成檔**
* 所有操作步驟一律寫在對應 Procedure 的 `Appendix` 中
* SOP 附錄變更 **不視為主文件改版**

目的在於：

* 控制文件數量
* 降低核准與改版成本
* 提高長期可維護性

---

### 3. 最小文件集合原則

本專案遵循以下原則：

* 能合併者不拆分
* 能用 Procedure 規範者，不另立 Policy
* 記錄優先由系統自動產生（如 AD、Wazuh、Veeam）
* 本專案 **不保存大量操作紀錄或日誌**

## 七、自動化工具

### 🚀 ISO 27001 自動化證據生成工具

本專案提供自動化工具，協助簡化證據文件的生成與管理：

**核心功能**：
- ✅ **模板引擎整合**：使用 Jinja2 自動填充模板
- ✅ **自動歸檔**：按日期自動歸檔至正確路徑 (`記錄與證據/{類別}/{YYYY}/{MM}/`)
- ✅ **合規性檢查**：掃描目錄結構並生成合規報告
- ✅ **週報生成**：從 Git Commit 歷史自動生成週報
- ✅ **GitHub Actions 整合**：自動化 CI/CD 流程

**快速開始**：

```bash
# 安裝依賴
cd scripts
pip install -r requirements.txt

# 列出所有模板
python iso_automation.py list-templates

# 生成證據文件
python iso_automation.py generate \
  --template "記錄與證據/備份與復原/備份執行紀錄_Template.md" \
  --data examples/backup_data.json

# 生成合規性報告
python iso_automation.py compliance-report --output compliance_report.md
```

**完整文檔**：
- 📖 [README.md](scripts/README.md) - 工具說明
- 📚 [USAGE_GUIDE.md](scripts/USAGE_GUIDE.md) - 使用指南
- 📁 [examples/](scripts/examples/) - 範例檔案
