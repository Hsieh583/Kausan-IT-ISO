## 停用檢查清單

每次停用帳號應完成以下檢查：

| 項目 | 狀態 |
|------|------|
| 通知帳號主人停用日期 | ☐ 已完成 |
| 檢查帳號是否有未轉交的權限或檔案 | ☐ 已完成 |
| 從 Active Directory 移除帳號 | ☐ 已完成 日期：_______ |
| 停用郵件帳號（設為非作用中） | ☐ 已完成 日期：_______ |
| 移除 VPN 與遠端存取權限 | ☐ 已完成 日期：_______ |
| 停用所有應用系統帳號 | ☐ 已完成 日期：_______ |
| 回收門禁卡 | ☐ 已完成 日期：_______ |
| 回收 VPN 令牌與裝置 | ☐ 已完成 日期：_______ |
| 確認帳號已完全停用（驗證登入失敗） | ☐ 已完成 日期：_______ |
| 記錄停用證明 | ☐ 已完成 日期：_______ |
