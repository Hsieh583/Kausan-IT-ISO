## 1. 目的

本指南規範ISO 27001記錄系統的結構、分類、存儲、檢索與管理，確保：
- 所有記錄完整、安全、易於查找
- 符合ISO 27001標準與法規要求
- 支持內部稽核、管理審查與持續改進
- 記錄生命週期管理（創建→保存→存取→銷毀）

## 3. 各類記錄的詳細說明

### 3.1 帳號與存取管理

#### 帳號申請與核准紀錄
- **位置**：`記錄與證據/帳號與存取管理/帳號申請與核准紀錄`
- **檔案格式**：Excel 工作表 或 PDF 掃描件
- **命名規則**：`帳號申請-YYYYMM-序號.xlsx`
  - 例：`帳號申請-202601-001.xlsx`
- **內容**：申請人、申請日期、系統、權限範圍、核准人、核准日期
- **頻率**：每個申請建立一筆紀錄
- **保存期限**：帳號停用後 5 年
- **責任單位**：IT部門
- **月度檢查**：審查未處理案件、逾期案件
- **分類等級**：內部使用

#### 帳號停用紀錄
- **位置**：`記錄與證據/帳號與存取管理/帳號停用紀錄`
- **檔案格式**：Excel 累計清單
- **命名規則**：`帳號停用清冊-2026.xlsx`（按年份）
- **內容**：帳號名稱、停用者、停用日期、停用理由、權限清除記錄
- **保存期限**：7 年
- **責任單位**：IT部門
- **月度檢查**：確認所有停用帳號已從系統清除
- **分類等級**：機密

#### 帳號權限審查紀錄
- **位置**：`記錄與證據/帳號與存取管理/帳號權限審查紀錄`
- **檔案格式**：Excel 報表 或 Word 文件
- **命名規則**：`帳號權限審查-YYYY-Q季度.xlsx`
  - 例：`帳號權限審查-2026-Q1.xlsx`
- **內容**：審查日期、審查範圍、帳號統計、異常發現、處置建議、審查人簽名
- **頻率**：每季審查一次
- **保存期限**：3 年
- **責任單位**：IT部門
- **分類等級**：機密

#### 特權帳號使用日誌
- **位置**：`記錄與證據/帳號與存取管理/特權帳號使用日誌`
- **檔案格式**：系統導出 CSV 或 SIEM 日誌
- **命名規則**：`特權帳號日誌-YYYYMMDD.csv`
  - 例：`特權帳號日誌-20260121.csv`
- **內容**：時間戳、帳號、使用者、操作命令、結果
- **頻率**：每日導出
- **保存期限**：1 年
- **責任單位**：IT部門
- **審查頻率**：每週审查異常操作
- **分類等級**：極密

### 3.2 密碼與認證管理

#### 密碼政策執行紀錄
- **位置**：`記錄與證據/密碼與認證管理/密碼政策執行紀錄`
- **檔案格式**：Word 驗證報告
- **命名規則**：`密碼政策驗證-YYYY年.docx`
- **內容**：驗證日期、系統清單、各系統密碼設定驗證結果、簽核
- **頻率**：每半年驗證一次
- **保存期限**：1 年
- **責任單位**：IT部門、資安部門
- **分類等級**：機密

#### 密碼重置紀錄
- **位置**：`記錄與證據/密碼與認證管理/密碼重置紀錄`
- **檔案格式**：Excel 清單
- **命名規則**：`密碼重置-YYYYMM.xlsx`
- **內容**：申請日期、申請人、帳號、重置日期、驗證方式、臨時密碼發放記錄
- **頻率**：每次重置建立一筆紀錄
- **保存期限**：1 年
- **責任單位**：IT部門
- **月度檢查**：統計重置頻率，識別異常帳號
- **分類等級**：極密

#### 密碼庫存取與審計日誌
- **位置**：`記錄與證據/密碼與認證管理/密碼庫存取日誌`
- **檔案格式**：密碼庫系統導出 CSV/PDF
- **命名規則**：`密碼庫審計-YYYYMM.csv`
- **內容**：存取時間、存取人、存取帳號、操作(查看/借出/歸還)、結果
- **頻率**：每月導出審計報告
- **保存期限**：1 年
- **責任單位**：IT部門、資安部門
- **分類等級**：極密

### 3.3 資產管理

#### IT資產清冊
- **位置**：`記錄與證據/資產管理/IT資產清冊`
- **檔案格式**：Excel 主檔案（實時更新）
- **檔名**：`IT資產清冊-2026.xlsx`（按年份）
- **內容**：資產ID、名稱、分類、擁有者、取得日期、成本、生命週期狀態、位置
- **更新頻率**：資產變更時隨即更新；月度檢查
- **保存期限**：資產報廢後 3 年
- **責任單位**：IT部門
- **分類等級**：內部使用

#### 資產分類表
- **位置**：`記錄與證據/資產管理/資產分類表`
- **檔案格式**：Excel 清單
- **檔名**：`資產分類表-202601.xlsx`
- **內容**：資產ID、分類等級(公開/內部/機密/極密)、標示方式、分類日期、分類人
- **更新頻率**：每季審查一次
- **保存期限**：長期保存
- **責任單位**：IT部門、資安部門
- **分類等級**：機密

#### 資產盤點紀錄
- **位置**：`記錄與證據/資產管理/資產盤點紀錄`
- **檔案格式**：Excel 盤點報告
- **命名規則**：`資產盤點-2026-Q季度.xlsx`
  - 例：`資產盤點-2026-Q1.xlsx`
- **內容**：盤點日期、範圍、清冊資產數、實際資產數、差異、原因分析、處置
- **頻率**：每季一次
- **保存期限**：3 年
- **責任單位**：IT部門、總務部門
- **分類等級**：內部使用

#### 資產維護與保養紀錄
- **位置**：`記錄與證據/資產管理/資產維護與保養紀錄`
- **檔案格式**：Excel 清單 或 維修廠商報告
- **命名規則**：`資產維護-YYYYMM.xlsx`
- **內容**：資產ID、維護日期、類型(保養/故障/升級)、廠商、費用、簽署
- **頻率**：每次維護建立紀錄
- **保存期限**：設備生命週期期間 + 3 年
- **責任單位**：IT部門
- **分類等級**：內部使用

### 3.4 安全事件與監控

#### 資訊安全事件日誌
- **位置**：`記錄與證據/安全事件與監控/資訊安全事件日誌`
- **檔案格式**：Wazuh 平台導出 CSV
- **命名規則**：`安全事件日誌-YYYYMM.csv`
- **內容**：事件ID、時間、類型、嚴重性、受影響系統、初步描述
- **頻率**：每日導出；實時監控
- **保存期限**：3 年
- **責任單位**：IT部門、資安部門
- **日常檢查**：每日審查高風險事件
- **分類等級**：機密

#### 事件調查與分析紀錄
- **位置**：`記錄與證據/安全事件與監控/事件調查與分析紀錄`
- **檔案格式**：Word 報告 或 PDF
- **命名規則**：`事件調查-事件ID-YYYYMMDD.docx`
- **內容**：事件編號、調查開始日期、調查人員、根因分析、受影響範圍、證據、改善措施
- **頻率**：每個 Major/Critical 事件產生一份報告
- **保存期限**：5 年
- **責任單位**：資安部門
- **分類等級**：極密

#### 系統日誌與事件日誌
- **位置**：`記錄與證據/安全事件與監控/系統日誌與事件日誌`
- **檔案格式**：SIEM 系統日誌 (Wazuh/ELK)
- **保存期限**：
  - 標準日誌：1 年
  - 核心系統 (AD, DB, 檔案伺服器)：2 年
  - 安全事件：3 年
- **責任單位**：IT部門
- **月度檢查**：驗證磁碟空間、自動輪轉、備份完成
- **分類等級**：極密

#### 變更管理紀錄
- **位置**：`記錄與證據/安全事件與監控/變更管理紀錄`
- **檔案格式**：Excel 變更申請單 或 變更管理系統導出
- **命名規則**：`變更單-CHG-YYYYMMDD-序號.xlsx`
- **內容**：變更編號、日期、申請人、影響系統、變更內容、風險評估、核准、執行紀錄、回復計畫
- **頻率**：每個變更建立一筆紀錄
- **保存期限**：3 年
- **責任單位**：IT部門
- **月度檢查**：統計變更數、失敗率、重新部署情況
- **分類等級**：機密

### 3.5 備份與復原

#### 備份執行紀錄
- **位置**：`記錄與證據/備份與復原/備份執行紀錄`
- **檔案格式**：Veeam 備份軟體導出報表
- **命名規則**：`備份日誌-YYYYMMDD.csv`
- **內容**：備份對象、備份類型、大小、耗時、成功/失敗、失敗原因
- **頻率**：每日導出（備份完成後）
- **保存期限**：1 年
- **責任單位**：IT部門
- **日常檢查**：每日驗證備份成功狀態
- **分類等級**：機密

#### 備份還原測試紀錄
- **位置**：`記錄與證據/備份與復原/備份還原測試紀錄`
- **檔案格式**：Word 測試報告
- **命名規則**：`還原測試-2026-Q季度-序號.docx`
- **內容**：測試日期、備份來源、還原目標、測試流程、結果(成功/失敗)、預計RTO、簽署
- **頻率**：每月至少一次；關鍵系統每週一次
- **保存期限**：3 年
- **責任單位**：IT部門、備份管理員
- **分類等級**：機密

#### 異地備份紀錄
- **位置**：`記錄與證據/備份與復原/異地備份紀錄`
- **檔案格式**：Excel 清單
- **命名規則**：`異地備份-YYYYWW.xlsx`（按年度與週次）
- **內容**：備份日期、檔案清單、大小、傳送方式、驗證結果、完整性檢查、遠端確認
- **頻率**：每週一次（通常周五晚間）
- **保存期限**：1 年
- **責任單位**：IT部門、備份管理員
- **分類等級**：極密

### 3.6 漏洞與補丁管理

- **命名規則**：`修補部署-2026-月份.xlsx`

- **頻率**：每次修補部署建立紀錄
- **保存期限**：2 年
- **責任單位**：IT部門
- **月度檢查**：驗證 Critical/High 修補部署完成率
- **分類等級**：機密

- **責任單位**：資安部門
- **分類等級**：機密

- **分類等級**：極密

### 3.8 合規與稽核

#### 內部稽核報告
- **位置**：`記錄與證據/合規與稽核/內部稽核報告`
- **檔案格式**：Word 報告 + Excel 工作紙
- **命名規則**：`內部稽核-AUD-YYYY-Q季度-序號.docx`
- **內容**：稽核計畫、稽核日期、被稽核單位、發現事項(符合/不符合/觀察)、證據、改善建議、簽署
- **頻率**：依年度計畫，通常每季 1-3 次稽核
- **保存期限**：3 年
- **責任單位**：資安部門、稽核人員
- **分類等級**：機密

#### 管理審查紀錄
- **位置**：`記錄與證據/合規與稽核/管理審查紀錄`

## 4. 記錄生命週期管理

### 4.1 記錄的四個階段

```
┌─────────────┬──────────────┬──────────────┬──────────────┐
│   建立      │   保存       │   存取       │   銷毀       │
├─────────────┼──────────────┼──────────────┼──────────────┤
│ • 按規定    │ • 電子/紙本  │ • 僅授權人   │ • 到期刪除   │
│   產生      │ • 備份       │ • 存取日誌   │ • 見證銷毀   │
│ • 完整準確  │ • 加密保護   │ • 不可複製   │ • 保存證明   │
│ • 及時簽署  │ • 防火環保   │ • 定期審查   │ • 3年存檔    │
└─────────────┴──────────────┴──────────────┴──────────────┘
```

### 4.2 詳細步驟

**建立階段**
1. 依記錄定義的頻率、格式、內容產生記錄
2. 確保完整性、準確性與及時性
3. 由負責單位人員簽署確認
4. 賦予分類等級與編號

**保存階段**
1. 電子記錄：存放於共享網路磁碟（如 `\\fileserver\ISO27001_Records\`）
2. 紙本記錄：存放於防火防濕檔案室
3. 執行定期備份（電子記錄日備份；紙本記錄年度複印備份）
4. 重要記錄加密存儲、異地保存

**存取階段**
1. 設定檔案權限：只有授權人員可存取
2. 記錄所有存取操作（誰、何時、存取何檔案）
3. 禁止下載複製；需查閱者在審批下於指定位置查閱
4. 定期審查存取記錄，異常即刻調查

**銷毀階段**
1. 到期記錄根據保存期限清單進行銷毀
2. 電子記錄：永久刪除或清除安全覆寫
3. 紙本記錄：碎紙機或焚化（需見證簽署）
4. 保存銷毀證明（含日期、數量、見證人）≥3年

## 5. 記錄存儲標準

### 5.1 電子記錄存儲

**共享網路磁碟結構**
```
\\fileserver\ISO27001_Records\
├── 2026年
│   ├── 01月
│   │   ├── 帳號管理
│   │   ├── 密碼管理
│   │   ├── 資產管理
│   │   └── ...
│   ├── 02月
│   └── ...
├── 2025年
└── 2024年
```

**檔案命名規則**
- 格式：`[類別]-[YYYYMM]-[序號]_[簡述].檔案格式`
- 例：`帳號申請-202601-001_新進員工王小明.xlsx`
- 例：`備份日誌-20260121.csv`

**存儲要求**
- 備份：每日備份；重要檔案每更新即備份
- 加密：機密與極密記錄存儲時加密（BitLocker/VeraCrypt）
- 版本控制：啟用共享磁碟版本控制功能，保留最近10個版本
- 存取日誌：所有存取操作自動記錄（Wazuh 監控）

### 5.2 紙本記錄存儲

**存儲位置**
- 防火防濕檔案室（資訊部內）
- 環境：15-25°C，濕度 30-70%
- 定期檢查：每季檢查一次，確認無受潮、蟲害

**組織方式**
- 按類別與年度分檔
- 檔案盒標籤清晰可見
- 建立檔案目錄清單
- 借閱時簽署領取、歸還簽證

**備份複印**
- 重要紙本記錄（如政策核准簽署、稽核報告）應複印一份
- 複印備份存放於另一安全地點
- 年度清點，更新備份
